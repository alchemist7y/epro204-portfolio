## Publish on GitHub Pages (free)
1. **Create a new repository** on GitHub (e.g., `epro204-portfolio-puja`).
2. **Upload** these three files to the root of the repo:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Go to **Settings ‚Üí Pages**:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (or `master`) / **Folder:** `/ (root)`
   - Click **Save**.
4. Wait ~30‚Äì60 seconds. Your site will be live at the URL shown under **Pages** (something like `https://YOUR-USERNAME.github.io/epro204-portfolio-puja`).
